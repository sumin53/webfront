문자열 속성 선택자

        속성값의 문자열을 확인하여 스타일을 적용하는 방식의 선택자.
        
        선택자[속성 ~= "특정값"] {css 코드;}
        -> 띄어쓰기로 구분되어있는 여러 속성값이 작성된 속성중
           속성값이 특정 값을 단어로 포함하는 태그 선택
        
        선택자[속성 |= "특정값"] {css 코드;}
        -> 속성값이 특정값을 단어로 포함하는 태그를 선택한다
            + "-" 기호로 구분,"-" 앞의 내용이 동일해야한다.
        
        --> 여러 속성중 맨 앞이면서,
            "-" 기호로 구분, "-" 앞의 내용이 동일해야한다.
            "-" 기준 앞쪽 문자열이 특정 값과 같은 태그 선택

        선택자[속성 ^= "특정값"] {css 코드;}
        -> 속성값이 특정 값으로 시작하는 태그 선택

        선택자[속성 $= "특정값"] {css 코드;}
        -> 속성값이 특정값으로 끝나는 태그 선택

        선택자[속성 *= "특정값"] {css 코드;}
        -> 속성값이 특정 값을 포함하는 태그
