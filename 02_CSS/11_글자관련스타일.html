color 속성 : 글자색을 지정하는 속성

        선택자 { color : 색상명(영문) | 16진수 숫자(#fff,#ffffff) | rgb값 | rgba값 | hsl값 | hsla값 }

        rgb (빛의 삼원색) : rgb(0~255, 0~255, 0~255 );
        hsl (색조, 채도, 밝기) : hsl(0~255, 0~255, 0~255 );

  • 16진수 값으로 지정
