목록 관련 태그

li 태그 (List Item) : ul, ol 태그 내부에 사용되는 목록 내용 태그

ul 태그(Unordered List): 순서가 없는 목록 (영역)

  • java
  • db(oracle)
  • jdbc
  • html5

ol 태그(Ordered List) : 순서가 있는 목록 (영역)

            type 속성 : 순서를 나타내는 형식을 변경하는 속성
            "a" : 영어 소문자
            "A" : 영어 대문자

            "i" : 로마 소문자
            "I" : 로마 대문자

            start 속성 : 순서 시작을 지정하는 속성
            reversed 속성 : 역순

  2. db(oracle)
