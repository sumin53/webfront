영역 관련 태그


        block 형식
        - 공간을 수직 분할하는 영역
          (수직으로 되어있는 공간을 수평으로 자르는것 -> 수직선 상에 영역들이 존재)
        
        - width, height 사용 가능
        - h1 ~ h6, p, pre, hr, div 등

        inline 형식
        - 공간을 수평 분할하는 영역
          (수평으로 되어있는 공간을 수직으로 자르는것 -> 수평선 상에 영역들이 존재)
        
        - width, height 사용 불가
        - b, strong, i, em, mark, span 등

block, inline 차이점 1: 줄 바꿈

div 태그 (division, 나누기) : 화면 영역을 나누는(분할하는) 용도(block)

첫 번째 영역
두 번째 영역
세 번째 영역

block 형식 태그는 이미 존재하는 태그의 다음줄 영역이 설정된다

span 태그(간격) : 줄바꿈이 일어나지 않고 옆으로 다음 영역이 설정된다.(inline)

첫 번째 영역 두 번째 영역 세 번째 영역

block, inline 차이점2 : 영역 지정 방식

        block 형식 : 사각형 박스 형태로 한 줄 전체를 영역으로 지정
        lnline 형식 : 내용(Content) 단위로 영역을 지정
