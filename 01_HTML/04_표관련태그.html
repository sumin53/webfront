표 관련 태그

table 태그 (영역)

        웹 문서에서 자료정리는 위해 주로 사용하는 태그
        행과 열로 이루어져 있으며, 행과 열이 교차하는 지점을 셀(cell)이라고 함

tr 태그 (table row) : 테이블의 한 행을 나타내는 태그(영역)

td 태그 (table date) : 한 셀을 나타내는 태그(영역 or 내용)

th 태그 (table header) : 행이나 컬럼의 제목을 나타내는 태그

caption 태그 : 테이블의 제목이나 설명을 추가하는 태그

table 관련 속성

        1) td, th 사용 가능 속성
        - rowspan : 행 병합 == 위, 아래 셀 합치기
        - colspan : 열 병합 == 좌, 우 셀 합침
        - width : 폭(넓이)를 지정하는 속성
        - height : 높이를 지정하는 속성

        2) table 사용 가능 속성
        - border : table,td, th에 테두리 추가

table 구조 설정 태그

        thead 태그 : 테이블의 상단 부분 영역 (컬럼명)
        tbody 태그 : 테이블의 중단 부분 영역 (실제 값, 내용)
        tfoot 태그 : 테이블의 하단 부분 영역 (합계, 누적)
