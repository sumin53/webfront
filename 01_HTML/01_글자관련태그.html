글자 관련 기타 태그

b태그 (bold) : 단순히 글자를 굵게 표시하는 태그
strong 태그 : 글자를 강조하는 태그 (웹 접근성)
i 태그(italic) : 단순히 글자를 기울이는 태그
em 태그 (emphasized text) : 글자를 강조하는 태그(웹 접근성)
mark 태그 : 형광펜 효과를 나타내는 태그
u 태그 (under line) : 밑줄
s 태그 : 취소선
sup 태그 : x2
sub 태그 : log2
abbr 태그 (abbreviation, 약어) : 마우스 오버시 툴팁 형식으로 설명 출력하는 태그
