시각적 효과를 위한 Effect 메소드

        요소에 애니메이션 효과를 만들기 위한 메소드 집합

        1. show() / hide()
        2. fadeIn() / fadeOut()
        3. slideDown() / slideUp()

        * effect 메소드 작성 방법

        1. $(‘선택자’).메소드명();
	    2. $(‘선택자’).메소드명(speed);
	    3. $(‘선택자’).메소드명(speed, easing, callback);

	    speed : 실행속도(밀리세컨드초) / 숫자 or slow, fast
	    easing : 변경되는 지점별 속도 / linear, swing 가능
	    callback : 메소드 실행 후 실행할 함수

1. show() / hide()

        show() : 요소를 점점 커지게 하면서 화면에 보여짐
        hide() : 요소를 점점 작아지게 하면서 화면에서 숨김

2. fadeIn() / fadeOut()

        fadeIn() : fade : 자취를 감추다, 사라지다 In : 안으로 들어옴
                   점점 불투명(선명) 해지면서 나타남
        fadeOut() : 점점 투명해지면서 사라짐

3. slideDown() / slideUp() 

        slideDown() : 아래로 내려가면서 보여짐 (펴짐)
        slideup() : 위로 올라가면서 없어짐 (접힘)

토글(toggle) 버튼 동작 만들기

        토글(toggle)이란?
        - 하나의 버튼에 두가지 상태를 지정할 수 있는 버튼
          ex) 전등 스위치

        toggle() 메소드
        - 메소드 수행 시 요소의 display 상태에 따라서
          자동으로 show(), hide()를 수행하는 메소드
