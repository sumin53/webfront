input 태그 관련 이벤트

        focus : input 태그가 선택되어 있는 경우
                ex) text 타입 input 태그에 커서가 깜박거리는 경우

        blur : focus를 잃었을때
                ex ) text 타입 input 태그에 커서가 깜박거리는 중 다른 곳을 클릭했을 때

        change : input 태그 값(value) 또는 체크 여부가 변했을때
            
            1) checkbox, radio, select 
                -> 값이 변하자 마자 change 이벤트가 발생
            
            2) text 관련 input 타입(text, email, tel, url, search, number 등)
                -> 입력과 동시에 값이 변했다고 인지하는 것이 아니라
                   포커스를 잃은 경우에 이전 값과 현재 값이 다르면
                   change 이벤트가 발생했다고 여김

        select : 요소 입력 영역에 블럭이 설정된 경우
