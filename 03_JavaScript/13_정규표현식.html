정규 표현식(Regular Expression)

        특정한 규칙을 가지는 문자열의 집합을 표현하는 사용하는 언어

        정규 표현식을 사용하면 입력된 문자열에 대한
        특정 조건 검색, 일치여부, 치환에 대한 조건문을 간단히 처리할 수 있다.

정규 표현식 참고 사이트

  • 작성한 정규표현식을 그림으로 나타내주는 사이트
  • 정규표현식 테스트 사이트
  • MDN - 정규표현식 설명

정규 표현식 객체 생성 및 확인 메소드

        1. 정규 표현식 객체 생성 방법
            1) const regExp = new RegExp("정규표현식");
            2) const regExp = /정규표현식/;
                -> 양쪽 '/' 기호가 정규 표현식의 리터럴 표기법

        2. 문자열 패턴 일치 여부 확인 메소드
            1) regExp.test(문자열)
                - 문자열에서 정규표현식과 일치하는 패턴이 있는 경우 true, 없으면 false (조건식)

            2) regExp.exec(문자열)
                - 문자열에서 정규표현식과 일치하는 패턴이 있는 경우
                  처음 매칭되는 문자열을 반환
                  없으면 null 반환 (문자열이 있고 없고의 문제이기 때문에 if문)

정규 표현식의 메타 문자

        문자열의 패턴을 나타내는 특수문자
        특수문자마다 지정된 특별한 뜻이 담겨있다.

        ^ (캐럿) : 문자열의 시작을 의미 
        $ (달러) : 문자열의 끝을 의미

        \w (word, 단어) : 아무단어 , 아무글자(글자수 제한x, 띄어쓰기x, 한글x, 특수문자x)
        \d (digit,숫자) : 0~9 사이 숫자 중 하나
        \s (space, 공간) : 공백 문자(띄어쓰기,엔터,tab 등등)
        ㄱ-힣 : 한글 모두(자음, 모음, 받침 관계없이 모두)
        \_ : "_" 문자 자체를 나타내는 이스케이프 문자 (\w에 포함됨.)
        * : 0개 이상 (없거나, 여러개가 있을 수 있음)
    
        [abcd] : 문자열 내에 a,b,c,d 중 하나라도 일치하는 문자가 있는지 검색
        a : 문자열 내에 a라는 문자가 있는지 검색 

        ex) /^a/ : a로 시작하는 문자열
            -> aaa,abc, apple (o), zzz, ppp, 가나다(x)

문자열이 j 또는 a로 시작하고, t또는 x로 끝나는지 확인

추가 메타 문자

        * : 0개 이상 == 0번 이상 반복
        + : 1개 이상 == 1번 이상 반복
        ? : 0개 또는 1개 (있으면 1개만 있음)
        . : 개행 문자를 제외한 모든 단일 문자(문자 한칸)

        수량 관련문자

        a{5}   : a가 5개 == aaaaa
        a{2,5} : a가 2개 이상 5개 이하 == aa ~ aaaaa
        a{2,}  : a가 2개 이상 == aa ~
        a{,5}  : a가 5개 이하

주민등록번호 검사

주민등록번호 입력("-" 포함)

flag 문자

        정규표현식 객체 뒤에 붙이는 문자

        g : 문자열 전체를 검색(전역 비교)
        i : 대소문자를 가리지 않고 비교
        m : 여러 줄의 검사를 수행
