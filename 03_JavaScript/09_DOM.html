DOM(Document Object Model)

        html(웹문서, 웹페이지) 문서를 객체 기반으로 표현한 것
        -> HTML 문서에 작성된 내용을 트리구조(계층형으로 표현)로 나타냈을 때
           각각의 태그, TEXT, COMMENT 등을 Node라고 한다. 

        document : {
            DOCTYPE : html,
            HTML : {
                HEAD : {
                    TITLE : "문서 제목",
                    STYLE : {..},
                    META : {..}
                },
                BODY : {
                    H1 : {TEXT : "제목", ATTRIBUTE : "속성"},
                    COMMENT : {TEXT : "주석 내용"},
                    DIV : {...}
                }
            }
        }

Node와 Element의 차이

        Node : 태그, 속성, 주석, 내용, 띄어쓰기 등을 모두 노드라고 표현함.
        Element : Node의 하위 개념으로 요소 노드만을 가리킨다.
