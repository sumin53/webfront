이벤트(Event)

        이벤트(Event) : 동작, 행위
            ex) 클릭(click), 마우스 오버(mousover), 키다운 (keydown), 키업(keyup) ...

     
        이벤트 리스너(Event Listener) : 동작, 행위를 감지( ~~~했을 때)
                                     + 동작 발생 시 연결된 기능을 수행

            - on + Event(행위) 붙여서 부름.
            ex) onclick(클릭 했을때), onmouseover(마우스가 올라왔을떄) ...


        이벤트 핸들러(Event Handler)
            - 이벤트 리스너에 연결된 기능(함수)
            - 이벤트 발생 시 수행하고자 하는 내용을 작성해둠.

인라인 이벤트 모델

        html 태그에 직접 이벤트(리벤트 리스너)를 작성하는 방법
        on이벤트명 = 함수명 (); 형태

고전 이벤트 모델

        요소가 가지고 있는 이벤트 속성에 이벤트 핸들러는 연결하는 방법
        인라인 모델과 다르게 script로만 작성 할 수 있음.

고전 이벤트 모델 활용(복습)

이벤트 발생 객체

        이벤트 발생 객체는 이벤트 핸들러의 매개변수로 이벤트 정보를 전달함.
        (매개변수명 e 또는 event)

고전 이벤트 모델의 문제점

        고전 이벤트 모델(요소.on이벤트)은 하나의 요소에 
        같은 이벤트리스너에 대한 동작을 여러개 지정할 수 없다.
        -> 여러 동작을 지정한 경우 마지막 내용만 적용된다.

표준 이벤트 모델(addEventListener("이벤트명",이벤트 핸들러(함수)))

        - W3C에서 공식적으로 지정한 이벤트 모델 (표준)

        - 고전 이벤트 모델의 단점인 하나의 리스너에 하나의 핸들러만 설정하는 것을 보완.

        - IE는 9버전 이상부터 사용 가능
