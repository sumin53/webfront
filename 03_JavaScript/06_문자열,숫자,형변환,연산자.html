문자열

        "",''를 리터럴 기호로 사용하는 값
        (js는 char 자료형이 없다!)

        String 자바스크립트 내장 객체 제공
        해당 객체에는 문자열을 위한 기본적인 함수들을 제공함.

숫자(number), 숫자 관련 함수

        숫자 자료형(number)의 리터럴 표기법
        - 10(정수), 3.14(실수), Infinity(무한), NaN(Not a Number)

        Math : 숫자 관련 함수를 제공하는 자바스크립트 내장 객체

문자열 -> 숫자 형변환

        숫자로만 이루어진 문자열  ex "123.456"
        정수, 실수 형태로 변환하는 방법이 존재함.

        parseInt("숫자로된 문자열") -> 정수 숫자로 변환 ex) 3.14 > 3
        parseFloat("숫자로된 문자열") -> 정수 -> 정수 / 실수 -> 실수 ex) 3.14 > 3.14
        Number("숫자로된 문자열") -> 정수 -> 정수 / 실수 -> 실수 ex) 3.14 > 3.14

동등 / 동일 비교 연산자

        동등 비교 연산자 ( ==, != )
        -> 자료형에 관계 없이 값이 같으면 true 반환
        ex) 1 == "1" -> true

        동일 비교 연산자 ( ===, !== )
        -> 자료형 , 값이 모두 같으면 true 반환
        ex) 1 === "1" -> false ( 자료형이 다름 )
