자바스크립트의 객체

        자바스크립트의 객체는 {} 내에 Key:Value가 모여있는 형태로 작성된다.
        (Map 형식)
        {k:v, k:v, k:v}
        
        (참고)
        자바스크립트 객체 모양으로 작성된 문자열
        == JSON(JavaScript Object Notation, 자바스크립트 객체 표기법)
        "{k:v, k:v, k:v}"

        - 객체를 생성 방법
        1) {} (객체 리터럴 표기법)을 이용한 생성 방법
        2) 생성자 함수 + new 연산자를 이용한 생성 방법

생성자 함수

        - 객체 생성 시 호출하는 함수

        - 일반적인 함수와는 다르게
          new 연산자를 이용해 객체를 생성해야 한다.
